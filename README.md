# PQ 产品一致性检测分析系统

## 1. 项目简介

本项目用于对多组 CSV 工况测试数据进行自动化分析，生成包含以下内容的完整检测报告：

* 产品 PQ 曲线分析（压差-流量特性）
* 迟滞（Hysteresis）分析
* 产品一致性分析（Max/Min/Avg/Deviation）
* **多模式平均值计算（3种可选判定标准）**
* 加权一致性影响评估（产品优劣排序）
* 去程 / 回程 PQ 曲线分析
* 多产品 PQ 对比图
* Excel 自动报告生成（含嵌入图表）
* PNG 图形自动输出

系统适用于：

* 电磁阀 / 比例阀 / 液压阀测试数据分析
* 多文件批量性能对比
* 工况一致性评估与质量分级

## 5. 核心功能说明

### 5.1 数据处理流程

1. 自动扫描 CSV 文件
2. 提取：
   * 电流值
   * 流量值
   * 压差值
3. 数据清洗（NaN / 非法值过滤）
4. 按电流分组滤波（仅压差）
5. 前50% / 后50% 分离（去程 / 回程）

---

### 5.2 PQ曲线分析

* 按标准流量点匹配：
```python
REF_FLOW = [5, 10, 20, 40, 60]
```
* 自动生成：
  * 单产品 PQ 曲线
  * 多产品 PQ 对比曲线
  * 去程 PQ
  * 回程 PQ
  * 去程+回程合并曲线

---

### 5.3 迟滞分析

计算方式：
```
Hysteresis = |Forward - Backward|
```
输出：
* 产品迟滞曲线图
* Excel迟滞表
* 超标自动染色（红色等级）

---

### 5.4 一致性分析（支持3种平均值计算模式）

**核心升级：支持 3 种平均值判定方式，满足不同检测标准**

1. **模式 0：原始模式**
   平均值 = (最大值 + 最小值) / 2
2. **模式 1：全体真实平均值（推荐）**
   平均值 = 所有产品实际数据的算术平均值
3. **模式 2：指定基准文件模式**
   平均值 = 选定基准产品的实测值

核心指标：
* Max / Min / Avg
* 正偏差 / 负偏差
* 一致性百分比偏差

```
偏差% = (偏差值 / 平均值) × 100%
```

---

### 5.5 加权一致性（核心算法）

用于识别“影响整体性能的关键产品”。

特点：
* 按电流分级加权：
  * 0mA 权重最高
  * 300 / 600 / 900mA递减
* 关键流量点（20L/min）额外加权
* 输出：
  * 产品影响占比
  * 偏差排序
  * 推荐缩放系数（修正建议）

---

### 5.6 Excel自动报告

包含 4 个核心Sheet：

#### ① 原始数据
* 全部CSV解析结果

#### ② 迟滞数据
* 并自动插入 Scatter Chart

#### ③ 一致性数据
* Max / Min / Avg / Diff
* 支持3种平均值计算结果
* 自动生成一致性图表

#### ④ 加权一致性分析
* 产品影响排序
* 修正建议
* 0mA 60L/min 修正前后对比

## 6. 关键参数配置（CONFIG）

所有参数集中管理，无需改代码逻辑：

### 流量点
```python
REF_FLOW = [5, 10, 20, 40, 60]
```

### 一致性阈值
```python
CONSISTENCY_LIMIT_PCT = 5.0
```

### 一致性计算模式（新增）
```python
CONSISTENCY_MODE = 1
```
* 0 = 原始 (max+min)/2
* 1 = 全体真实平均值
* 2 = 指定基准文件

### 基准文件（模式2启用）
```python
BASE_FILE_NAME = "基准产品.csv"
```

### 滤波窗口
```python
FILTER_WINDOW = 201
```

### 核心流量
```python
KEY_FLOW = 20
```
